# Lijsten maken

Zoals hiervoor vermeld, kan je een nieuwe lijst aanmaken door **[]** te gebruiken.
```python
lijst1=[] #een lege lijst
lijst2=["huis","boom",35]
```

Je kan ook aan een bestaande lijst nieuwe elementen toevoegen met het commando **append()**
```python
lijst1=["huis","boom",35]

lijst1.append(67) 
#lijst1 wordt nu ["huis","boom",35, 67] 
```
