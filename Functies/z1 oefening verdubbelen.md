# 📝 Verdubbelen

## Opdracht
<div style="
    background-color: #b0cdff;
    padding: 15px;
    border-radius: 8px;
    border: 1px solid #ddd;
">Definiëer een functie *verdubbelen(x)* die een gegeven getal verdubbelt.
</div>
<div style="
    background-color: #daabff;
    padding: 15px;
    border-radius: 8px;
    border: 1px solid #ddd;
">ℹ️De naam van het argument *x* is niet belangrijk. Zorg dat je wel altijd voor een naam kiest die nergens anders in je code voorkomt.
</div>


### Voorbeeldtesten

Deze oefening kan je zelf uittesten door de functie **op te roepen.** Bijvoorbeeld: 'verdubbelen(3)', 'verdubbelen(70)', ...

| Oproepen functie  | Uitvoer |
| ------------- | ------------- |
|verdubbelen(3) | 6 |
