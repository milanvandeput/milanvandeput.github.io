# Sorteeralgoritmes
Naast het zoeken naar een specifieke waarde in een lijst, is het sorteren van een lijst getallen ook een veelgebruikte methode. Merk bijvoorbeeld op dat het binair zoeken een gesorteerde lijst vereist.

```python
getallenlijst=[11,7,28,4,30]
getallenlijstgesorteerd = [4,7,11,28,30]
```

Het sorteren van een lijst (van klein naar groot) kan op verschillende manieren gebeuren.

# Bubble sort

Bij *Bubble sort* wordt de volledige lijst doorlopen waarbij de elementen in paren worden vergeleken. Als hun volgorde onderling niet klopt, worden ze onderling gewisseld. 

Het algoritme blijft dit herhalen tot er bij het volledig doorlopen van de lijst geen enkele keer meer gewisseld moet worden.
