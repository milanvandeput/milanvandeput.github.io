# 📝 Start to Run
*Start to Run* is een trainingschema dat beginnende lopers kunnen volgen. De loper kiest een startafstand en een doelafstand. Elke dag loopt de loper 10% langer dan de dag ervoor, tot de doelafstand wordt bereikt.

Een voorbeeld: de startafstand is 6km en de doelafstand is 10km.
* Dag 1: 6km
* Dag 2: 6.6km
* Dag 3: 7.26km
* Dag 4: 7.986km
* Dag 5: 8.7846km
* Dag 6: 9.66km
* Dag 7: 10.62km <- het doel is bereikt.
