# Python code
Wij zullen code schrijven in de taal **Python** omdat deze gebruiksvriendelijk is en ook in zeer veel domeinen voorkomt. De code is ook redelijk leesbaar, zoals in het volgende voorbeeld:

```python.trinket
print("Hallo wereld")
```

Hier heb je kennis gemaakt met een eerste **functie**: de functie **print()**. Deze functie kan tekst of andere datastructuren *(zie later)* doen verschijnen op je scherm. Daarom is dit ook de belangrijkste functie want zonder *print()* zal je nooit kunnen zien wat je code aan het doen is.
