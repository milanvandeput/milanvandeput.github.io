# Opdracht
Schrijf een programma dat de waarden van de twee variabelen *a* en *b* omwisselt. *Tip: gebruik een derde variabele.*

```python
a=5
b=8
#vul deze code verder aan.
```


Controleer je code door de variabelen te printen met *print(a)* en *print(b)*.

# Uitdaging

Kan je hetzelfde doen zonder een derde variabele te gebruiken?
