# Tekstbestanden

De bestanden die we in deze les leren bewerken met python code, zijn tekstbestanden. Ze hebben de extensie **.txt** en kunnen geopend worden met programma's zoals **Kladblok**.

<a href="/assets/files/test.txt" download="download">Download hier een voorbeeld van een tekstbestand.</a>

In de oefeningen zullen we verschillende tekstbestanden inlezen en aanpassen. Daarvoor is het belangrijk dat het tekstbestand zich steeds in **dezelfde map**(*) bevindt als het *.py-bestand* (de python code). 

![Numeriek](/assets/images/tekst1.png)

*Een voorbeeld van hoe de map er kan uitzien. Er mogen ook meerdere tekstbestanden en .py-bestanden* in dezelfde map zitten.


(*) Het is ook mogelijk om in je code te verwijzen naar bestanden die zich in andere mappen bevinden. Daarvoor moet je het verschil kennen tussen absolute en relatieve paden, meer info: [^medium.com](https://medium.com/@balakrishna0106/understanding-the-difference-between-a-relative-and-absolute-path-in-python-dca68d84139a "Medium")
